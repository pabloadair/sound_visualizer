# Sound Visualizer - Android Audio Sync

Aplicación móvil para la visualización de audio y sincronización multimedia en tiempo real.

## Descripción
Aplicación desarrollada en Android Studio que genera animaciones visuales (burbujas y formas geométricas) que reaccionan dinámicamente al ritmo de la música.

## Funcionalidades Clave

Modo Concierto: Implementación de arquitectura Cliente-Servidor donde un dispositivo móvil actúa como servidor central, sincronizando las animaciones en pantalla de todos los dispositivos conectados al unísono.

Procesamiento de Audio: Análisis de frecuencias en tiempo real para la generación de gráficos.

## Tecnologías

Java/Kotlin (Android SDK).

Sockets para la comunicación en red local.
